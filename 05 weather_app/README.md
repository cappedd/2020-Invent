## Weather App
This app tells you the weather based on your location by making use of weather api generated by Open Weather Map.

### Functions Used
- [Try/Catch](https://dart.dev/guides/libraries/futures-error-handling)
- [Future](https://api.flutter.dev/flutter/dart-async/Future-class.html)
- [Weather API](https://openweathermap.org/api)
- [Status Codes](https://pub.dev/documentation/http/latest/http/BaseResponse/statusCode.html)
- [JSON](https://docs.flutter.dev/development/data-and-backend/json)

### Packages Used
- [Cupertino Icons](https://pub.dev/packages/cupertino_icons)
- [Geolocator](https://pub.dev/packages/geolocator)
- [http](https://pub.dev/packages/http)
